## To Get Started 
1. Download the Code
2. Open up Eclipse
3. Go to files -> Open Project
4. Choose **"haunted-university-basement"** not cs3330-hub

## To Build 
1. Right click on **"haunted-university-basement"**
2. Run As -> Install Maven or Set up
3. For Set up put this in the 'Goals' box:
   ```bash
   test-compile org.pitest:pitest-maven:mutationCoverage
   ```
4. Press Apply and then right click on the project
5. Find Maven Install
6. Then right click again then build
7. If you run into trouble clean then build

## While Working 
- While working and testing you DO NOT HAVE TO RE BUILD EVERYTIME you can right click and then **"Coverage As"** -> **"Junit Test"**
